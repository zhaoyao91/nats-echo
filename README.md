# NATS Echo

Echo NATS messages.

## Features

- easy to use
- compact and pretty display with colors
- support connecting to nats cluster

## Installation

```
npm i -g nats-echo
```

## Usage

```
nats-echo
```

## Options

- url = "nats://localhost:4222" - specify nats url
- topic = ">" - specify topic to subscribe
- printHeadLine = true
